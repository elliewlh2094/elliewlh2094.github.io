# 網站後續優化方向

這份文件記錄本輪維護優先重整之後的下一階段方向。排序依目前優先順序排列。

## 1. 頁面內容完整度

目標是讓首頁、關於頁、專案頁與文章頁的資訊密度更完整，能更清楚呈現研究方向、技術重點與代表作品。

優先項目：

- 首頁主敘事改成更清楚的技術焦點與代表作品導覽。
- 關於頁補強研究興趣、技術能力與網站用途。
- 專案頁補足問題背景、方法、取捨與成果。
- 文章頁補強來源、摘要、延伸閱讀與相關專案。

## 2. 視覺呈現

目標是讓網站在保持目前資訊清晰度的前提下，提升版面節奏與整體完成度。

優先項目：

- 強化首頁與列表頁的視覺層次。
- 重新檢查字級、留白與卡片比例。
- 統一封面圖片風格與替代文字品質。
- 補足桌面與手機的視覺一致性檢查。

## 3. 作品說服力

目標是讓公開專案從「有列出來」提升到「能說服面試官與同領域工程師」。

優先項目：

- 專案頁加入更完整的角色、背景、限制與結果敘事。
- 補齊正式封面與外部連結。
- 補強代表作品的重點摘要與延伸筆記。
- 規劃文章與專案的交叉連結策略。

## 這一輪不處理

- 搜尋、分類與系列文章功能
- 內容管理系統
- 大量新增新頁面
- 大規模品牌或定位改寫
