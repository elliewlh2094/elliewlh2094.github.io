## 核心目標

- 內容可用 Markdown 維護，讓研究筆記、副專案、文章與生活製作紀錄能低成本更新。
- 專案可用簡易圖文展示，讓讀者快速理解機器人研究、工具設計、技術重點與成果。
- 網站可作為長期筆記本，累積 AI 工具實作、機器人視覺筆記、遊戲心得與手作紀錄。
- UI 與功能維持分離，修改視覺設計時不影響內容模型、路由與草稿排除機制。
- 網站具備輕微極客與書呆子特質，讓研究內容保持可信，生活內容也能親近易讀。
